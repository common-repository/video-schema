== Installation ==

= Through Dashboard =
1. Go to Plugins -> Add New -> Search for "Video Schema" Or Upload the plugins zip file
= Through FTP =
1. Upload the plugin into `wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's It.
